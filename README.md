# SBTI 人格测试小程序

> 🎭 一款有趣的人格测试小程序，通过日常场景的选择，分析你的人格类型倾向

![license](https://img.shields.io/badge/license-MIT-blue.svg)
![platform](https://img.shields.io/badge/platform-WeChat-green.svg)

## ✨ 特性

- 🎯 **25道精选题目** - 涵盖工作、生活、社交等多个场景
- 🎭 **35种人格类型** - 摆烂联盟、奉献天团、敏感星人、佛系阵营、犬儒先锋等
- 🎲 **彩蛋类型** - 随机人、404人、BUG型等特殊人格
- 📸 **生成分享图** - Canvas 生成精美分享图片
- 🎨 **自定义导航栏** - 沉浸式设计体验
- 📱 **响应式设计** - 适配不同屏幕尺寸

## 📸 功能截图

### 首页
- 简洁大气的入口设计
- 清晰的测试信息展示
- 7种人格类型预览

### 测试页
- 实时进度显示
- 清晰的题目展示
- 便捷的选项交互

### 结果页
- 详细的人格分析
- 标签和关键词展示
- MBTI 类型对应
- 分享语展示

### 分享页
- Canvas 2D 绘图
- 精美的分享图片
- 一键保存到相册

## 🚀 快速开始

### 环境要求

- 微信开发者工具
- 微信小程序开发权限

### 安装步骤

1. **克隆仓库**
```bash
git clone https://gitee.com/wangfaqiang/SBTI.git
cd SBTI
```

2. **打开项目**
- 使用微信开发者工具打开项目文件夹

3. **配置 AppID**
- 在 `project.config.json` 中填入你的小程序 AppID

4. **编译运行**
- 点击"编译"按钮
- 在模拟器中预览

5. **真机调试**
- 点击"预览"生成二维码
- 微信扫码在真机上测试

## 📁 项目结构

```
SBTI/
├── pages/              # 页面文件
│   ├── index/         # 首页
│   ├── test/          # 测试页
│   ├── result/        # 结果页
│   └── share/         # 分享页
├── data/              # 数据文件
│   ├── questions.js   # 题库数据
│   └── types.js       # 人格类型数据
├── utils/             # 工具函数
│   ├── calculate.js   # 结果计算
│   └── system.js      # 系统工具
├── components/        # 组件
│   └── custom-navbar/  # 自定义导航栏
├── app.js             # 小程序入口
├── app.json           # 小程序配置
└── app.wxss           # 全局样式
```

## 🎯 核心功能

### 1. 测试流程
```
首页 → 测试页 → 结果页 → 分享页
```

### 2. 结果计算
- 根据用户选择累积各维度得分
- 计算得分最高的类型
- 彩蛋触发机制

### 3. 分享图生成
- 使用 Canvas 2D API
- 动态生成个性化分享图片
- 支持保存到相册

## 🎨 设计规范

### 颜色方案
- 主色：`#3b82f6`（蓝色）
- 成功色：`#22c55e`（绿色）
- 警告色：`#f59e0b`（橙色）
- 危险色：`#ef4444`（红色）

### 字体规范
- 标题：48rpx - 64rpx
- 正文：28rpx - 32rpx
- 辅助文字：24rpx - 26rpx

### 间距规范
- 页面边距：32rpx
- 卡片间距：16rpx - 24rpx
- 元素间距：12rpx - 16rpx
